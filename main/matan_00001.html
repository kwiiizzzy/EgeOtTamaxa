Каталог задач по математике:
В треугольнике ABC стороны AC и BC равны. Внешний угол при вершине B равен 107°. Найдите угол C
#00001
Назад
Если внешний угол треугольника при вершине B равен 107°, то при вершине B треугольника ABC равен 180° - 107° = 73°, так как это смежные углы.

Тогда, так как AC = BC:
Значит, по сумме углов треугольника:

∠A = ∠B = 73° => ∠A + ∠B = 146°
∠C = 180° - ∠A - ∠B = 180° - 146° = 34°
Ответ: 34
